# AgenticBro Bounty Agent — Cowork Instructions

This folder contains the AgenticBro autonomous scam detection agent built for the
OOBE Protocol + Ace Data Cloud bounty. Your job is to install it, configure it,
and get the autonomous workflow running.

## What this project is

A Node.js autonomous agent that:
- Registers an identity on-chain on Solana via the Synapse Agent Protocol (SAP)
- Calls 4 Ace Data Cloud APIs per cycle (web search, AI text gen, social, news)
- Settles payments on-chain via x402 after each cycle
- Loops continuously with no human input required

## Your task

Follow the HANDOFF_PLAN.md in this folder step by step. Do not skip steps.
Verify each step succeeded before proceeding to the next.

## Key facts

- Runtime: Node.js 20+ (check with `node --version`)
- Package manager: npm
- Config file: `.env` (copy from `.env.example`, then fill in keys)
- Wallet file: `wallet.json` (generated by setup script — do NOT commit this)
- Two API keys needed (both free): Synapse RPC + Ace Data Cloud
- Main command to run: `npm run workflow:loop`
- Dashboard: `npm run dashboard` → opens at http://localhost:3333

## Do not

- Delete wallet.json
- Commit .env or wallet.json to any repo
- Skip the `npm run register` step — agent must be on-chain before workflow runs
- Run `npm run workflow:loop` before registration is confirmed

## If something fails

Check error output carefully. Most failures are one of:
1. Missing .env key (SYNAPSE_RPC_URL or ACE_DATA_API_KEY not set)
2. Wallet not funded (needs ~0.1 SOL)
3. Node version too old (needs 20+)
4. npm install not run yet
